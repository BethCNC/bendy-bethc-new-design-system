Icon-only Button

Buttons use responsive tokens for padding, radius, border width, color, and font-size (icon scales with font-size).

  • Padding: p-[var(--ui-menu-item-v-padding)] (responsive via media queries)
  • Radius: rounded-button-radius-md (component token)
  • Border: border-[var(--ui-menu-item-border-width)] + border-border-neutral-dark
  • BG: bg-surface-neutral-card (or primary)
  • Icon size: inherits from font-size (e.g., text-body-md)
